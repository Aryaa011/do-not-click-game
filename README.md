# 🧠 Do Not Click

An interactive psychological web experience that plays with user behavior.

## 🎮 Features
- Progressive warning system
- Cursor-reactive UI
- Escape mechanics ("Catch me if you can")
- Dynamic visual transitions (calm → red mode)
- Custom cursor trail + glitch effects
- Sound-reactive interactions

## 🛠 Tech Used
- HTML
- CSS (animations, glitch effects)
- JavaScript (interaction logic)
